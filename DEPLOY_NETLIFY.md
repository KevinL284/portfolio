## 📋 **Como fazer o deploy (5 minutos):**

### **1. Criar conta no Netlify**
- Acesse: https://netlify.com
- Clique **"Get started for free"**
- **Importante:** Use o mesmo email do GitHub para facilitar

### **2. Conectar repositório**
- No dashboard, clique **"Add new site"** → **"Import an existing project"**
- Escolha **"Deploy with GitHub"**
- Autorize o Netlify a acessar seus repos
- Selecione o repositório **"portfolio"**

### **3. Configurar build**
- **Branch:** `main`
- **Build command:** deixe vazio (é só HTML estático)
- **Publish directory:** deixe vazio ou `.` (raiz)
- Clique **"Deploy site"**

### **4. Aguardar deploy**
- Primeira vez demora ~2-3 minutos
- Você receberá uma URL tipo: `https://amazing-name-123.netlify.app`

### **5. Configurar formulário (automático!)**
- ✅ **Nada a fazer!** O Netlify detecta automaticamente formulários
- ✅ Na primeira submissão, você receberá email de confirmação
- ✅ Depois todas as mensagens chegam no kevin.lucas284sz@gmail.com
